HOW TO RUN THIS APP
====================

=== FIRST TIME SETUP (do this only once) ===

STEP 1: Extract WinPython
  - In this folder you will find a file like "WinPython64-....exe"
  - Double-click it
  - When asked WHERE to extract, select "WinPython" in "pdf_password_converter" folder.
  - Click Extract. A new folder "WinPython" will appear.

STEP 2: Install required libraries
  - Double-click "install_dependencies.bat"
  - Wait until it finishes (you will see "DONE" at the end)
  - This only needs to be done once

=== RUNNING THE APP (every time) ===

  - Double-click "start_app.bat"
  - The app window will open
  - Done!

=== FIRST TIME USING THE APP ===

  - The app will ask you to add an employee
  - Enter initials (e.g. "KM") and a password
  - Then select a .docx or .pdf file
  - Click "Convert" button
  - The encrypted PDF will be saved in "Vyplatne Pasky" folder

=== MOVING TO ANOTHER COMPUTER ===

  - Copy the ENTIRE "pdf_password_converter" folder to USB
  - On the new computer, just double-click "start_app.bat"
  - It will ask to add employee again (passwords are stored per computer)

=== TROUBLESHOOTING ===

  Problem: "start_app.bat" shows error about python not found
  Solution: Make sure you did Step 1 (extract WinPython)

  Problem: App crashes with import error
  Solution: Run "install_dependencies.bat" again

  Problem: Nothing happens when double-clicking start_app.bat
  Solution: Right-click start_app.bat -> Run as administrator
