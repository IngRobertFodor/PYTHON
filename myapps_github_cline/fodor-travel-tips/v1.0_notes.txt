POPIS APLIKÁCIE:
================
Webová aplikácia na objavovanie bodov záujmu (POI) v oblasti,
kam cestujete. Kombinuje OpenStreetMap dáta, Wikipedia popisy,
počasie z Open-Meteo a relevatné linky (Reddit, AllTrails, Wikivoyage).

AKO APPKA FUNGUJE (FLOW):
==========================

1. POUŽÍVATEĽ ZADÁ DESTINÁCIU
   - Napíše názov mesta/miesta do vyhľadávania
   - ALEBO klikne priamo na mapu (Leaflet/OpenStreetMap)
   → Backend: /api/geocoding/search (Nominatim API)
   → Backend: /api/geocoding/reverse (pri kliku na mapu)

2. NASTAVÍ OKRUH HĽADANIA
   - Vzdialenosť v km (napr. 25 km)
   - ALEBO čas cestovania (napr. "1h 30min autom")
   → Backend: /api/geocoding/travel-radius (OSRM routing)
   → Frontend: zobrazí kruh na mape

3. VYBERIE DÁTUM CESTOVANIA
   - Date picker (dnes až +16 dní)
   - Pre predpoveď počasia v daný deň

4. KLIKNE "OBJAVIŤ MIESTA"
   → Frontend zobrazí loading
   
5. BACKEND SPRACUJE PARALELNE:
   a) POČASIE (/api/weather/grid)
      - Open-Meteo API
      - 3x3 grid bodov v oblasti (9 bodov)
      - Teplota, zrážky, vietor, UV, weathercode
      - Vyhodnotenie outdoor-friendly
      
   b) BODY ZÁUJMU (/api/poi/search)
      - Overpass API (OpenStreetMap databáza)
      - Filtruje 14 mainstream tagov + 21 alternatívnych tagov
      - Každý POI:
        → Skórovanie (Wikipedia prítomnosť, počet tagov, vzdialenosť)
        → Wikipedia API: popis + thumbnail
        → Wikidata API: oficiálny web
        → Links service: 5 linkov podľa typu POI

6. FRONTEND ZOBRAZÍ VÝSLEDKY:
   - Počasie v oblasti (center + 3x3 grid + summary)
   - 10 Mainstream POI kariet
   - 10 Alternatívnych POI kariet
   - Každá karta: meno, popis, vzdialenosť, typ, 5 linkov
   - POI markery na mape

TECHNOLÓGIE:
============
- Python Flask (backend server)
- HTML5 + CSS3 + Vanilla JavaScript (frontend)
- Leaflet.js (interaktívna mapa)
- Overpass API (OpenStreetMap POI dáta)
- Wikipedia REST API (popisy)
- Wikidata API (oficiálne weby)
- Open-Meteo API (počasie, 16 dní, zadarmo)
- Nominatim (geocoding, zadarmo)
- OSRM (routing/čas cestovania, zadarmo)

HOSTING:
========
- Render.com (free tier)
- Automatický deploy z GitHub
- Cold start: ~30s po 15 min nečinnosti

Spustenie testov:

Testové súbory:

  tests/test_geocoding_service.py - geocoding, reverse, travel radius

ZNÁME OBMEDZENIA v1.0:
=======================
- Overpass API timeout pri veľkom radiuse (>100km) v hustých oblastiach
- Wikipedia popisy nie vždy dostupné pre menej známe miesta
- Počasie max 16 dní dopredu
- Render.com free = cold start po 15 min
- Žiadne ukladanie histórie hľadania (plánované pre v1.1)
